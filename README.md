# Welcome to Study Buddy!

Study Buddy is a quiz which assesses the users learning style as visual, auditory, or kinesthetic. Once the user submits their quiz, they will be taken to a page in which they see their results including a blurb of their learning style and resource links (as well as written tips) catered to said learning style.

## What's Next?

While currently being tested as a front end in version 1, this project can look forward to growing as a full-stack application.
In the works are a personal page for the student (of which they will have their login information), a parent section, and a teacher section.
